# Days — Daily Accountability Logs

Each file follows the pattern `day-XXX.md` where XXX is the ordinal day count.

## Structure
- Contract Check (from previous day)
- What Got Done
- What Didn't Ship
- Habits & Patterns
- Self-Awareness Score (Execution, Focus, Energy, Self-Awareness — each 1-5)
- HT Score (CoS Score + CEO Score)
- One Line Truth
- Tomorrow's Contract (3 items max with "done when" criteria)

## Rules
- Only count actualized outcomes, not things that "moved forward"
- Log daily — gaps in the mirror are data too
- After 3+ days, surface patterns from the logs
