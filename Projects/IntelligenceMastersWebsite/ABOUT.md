# IntelligenceMastersWebsite

## What It Is
Main business portal for intelligencemasters.com — sells AI strategy, automation, and software services. Design philosophy: "radical simplicity" with an AI orb as the primary interface instead of traditional menus.

## Core Functionality
- Landing page featuring an AI orb (not traditional nav menus)
- Orb acts as a salesperson/consultant guiding users to services
- Backend handles consulting inquiries
- "Bounty board" system where clients submit tasks for the community to fulfill

## Priority & Status
- Core business website — represents the Intelligence Masters brand
- Existing codebase already in progress (see git history)

## Key Technical Notes
- Radical simplicity: orb-first, no traditional menus
- Orb is conversational — guides users to bounties/services
- Bounty board: clients post tasks, community fulfills them
- Built with Astro (based on existing repo)
