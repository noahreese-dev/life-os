# AFF - A Financial Friend

## What It Is
An agentic "orb" application that helps users achieve financial freedom through saving and investing. Acts as a financial consultant/mentor (personas like Alex Hormozi, Warren Buffett) rather than a passive tracker.

## Core Functionality
- Track user spending
- Create financial freedom plans
- Manage subscriptions (including using Playwright to physically navigate websites and unsubscribe users)
- AI-action-first interface: user speaks to the orb to initiate tasks

## Priority & Status
- High-priority "side quest" intended for monetization
- Interface paradigm: voice-driven orb interaction

## Key Technical Notes
- Agentic architecture (orb as the primary interface)
- Playwright integration for automated web actions (e.g. cancelling subscriptions)
- Persona system for different financial advisor styles
