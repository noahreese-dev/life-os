# Calendar Generator — Life-OS Chief of Staff

Generate a full visual calendar suite: HTML dashboard, print-ready PDF, and presentation slides.

## Steps

1. Read `STATUS.md`, `WEEKLY-LOG.md`, and `PRIORITIES.md` for context
2. Build the schedule based on priority rules and current project states
3. Generate THREE files:
   - `calendar.html` — Interactive dark-mode dashboard (opens in browser)
   - `calendar-slides.html` — Presentation deck (arrow-key navigation)
   - `calendar.pdf` — Print-ready PDF (generated via Edge headless from calendar.html)
4. Open the HTML in browser: `start "" "C:\Users\Pc\Desktop\Life-OS\calendar.html"`
5. Generate PDF: `"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --headless --disable-gpu --no-pdf-header-footer --print-to-pdf="C:\Users\Pc\Desktop\Life-OS\calendar.pdf" "file:///C:/Users/Pc/Desktop/Life-OS/calendar.html"`

## Design System (ALL outputs)

### Color Palette
- Background: `#0a0a0f` (near-black)
- Surface: `#12121a` (card backgrounds)
- Border: `#1e1e2e` (subtle separation)
- Text Primary: `#e2e8f0`
- Text Secondary: `#94a3b8`
- Accent glow: `rgba(139, 92, 246, 0.15)` (purple ambient)

### Project Colors
- Hardware Ad +: `#f97316` (orange)
- Garage33: `#22c55e` (green)
- UltrawashAPP: `#06b6d4` (cyan)
- IntelligenceMasters: `#8b5cf6` (purple)
- Meetings/Calls: `#eab308` (yellow)
- Life-OS/Admin: `#64748b` (slate)

### Typography
- Font: `'Inter', 'Segoe UI', system-ui, sans-serif`
- Headings: 600 weight, letter-spacing: -0.02em
- Body: 400 weight

### Visual Style
- Subtle glassmorphism on cards: `backdrop-filter: blur(12px)`, semi-transparent backgrounds
- Thin glowing borders on active/priority items
- Minimal shadows, rely on background contrast
- No gradients except subtle radial ambient glow behind key elements
- Rounded corners: 12px on cards, 8px on badges

## FILE 2: calendar-slides.html — Presentation Deck

Self-contained HTML presentation. No external dependencies. Arrow-key + click navigation.

### Slide Structure:
1. **Title Slide**: "Week of [Date Range]" — big, clean, centered. Subtle ambient glow.
2. **This Week's Mission**: 2-3 bullet priorities + total revenue at stake
3. **Monday** — Day theme + schedule blocks
4. **Tuesday** — Day theme + schedule blocks
5. **Wednesday** — Day theme + schedule blocks (highlight LB call if applicable)
6. **Thursday** — Day theme + schedule blocks
7. **Friday** — Day theme + schedule blocks (highlight deadlines)
8. **Key Dates & Deadlines** — Timeline visual
9. **Financial Overview** — Revenue table, $/hour breakdown
10. **Blockers & Risks** — What could go wrong, what needs decisions

### Slide Mechanics:
```html
<!-- Each slide is a full-viewport section -->
<section class="slide" id="slide-1">...</section>

<!-- CSS: slides stack, only active one visible -->
.slide {
  display: none;
  width: 100vw;
  height: 100vh;
  padding: 60px 80px;
}
.slide.active { display: flex; flex-direction: column; justify-content: center; }

<!-- JS: arrow key + click navigation -->
document.addEventListener('keydown', (e) => {
  if (e.key === 'ArrowRight' || e.key === 'ArrowDown') nextSlide();
  if (e.key === 'ArrowLeft' || e.key === 'ArrowUp') prevSlide();
});
```

### Slide Design:
- Full viewport slides with generous whitespace
- Large typography (titles: 48px, body: 24px, data: 20px)
- Slide counter in bottom-right: "3 / 10"
- Subtle transition between slides (opacity fade, 200ms)
- Progress bar at very top of viewport (thin 3px line, fills left to right)
